# タスクリスト: Cプリプロセッサ (CPP) 対応

このタスクリストは、C言語のプリプロセッサ機能の実装を管理するためのものです。
実装対象は `#include`, `#define`, `#undef`, `#if`, `#ifdef`, `#ifndef`, `#else`, `#elif`, `#endif` になります。

## 1. プリプロセッサ基盤の開発
- [x] `src/preprocess/preprocess.c` と `src/preprocess/preprocess.h` の新規作成
- [x] main パイプラインを `tokenize` -> `preprocess` -> `parse` に変更
- [x] トークン列を受け取り、展開後の新しいトークン列を返す `preprocess()` の枠組みを作成
- [x] ディレクティブ行（行頭が `#`）と通常のコード行を分離して走査するロジックの追加

## 2. インクルード機能 (`#include`)
- [x] `#include "ファイル名"` のパース対応
- [x] `#include <ファイル名>` のシステムインクルードパス探索対応
- [x] ファイル読み込みと `tokenize` の実行、メインのトークン列への挿入・マージ機能の追加
- [x] E2Eテストへのインクルード機能の追加・検証

## 3. オブジェクトマクロと定数置換 (`#define`, `#undef`)
- [x] マクロ定義を保存する辞書構造（ハッシュマップ等）の実装
- [x] `#define 識別子 置換リスト` のパースと辞書への登録
- [x] `#undef 識別子` のパースと辞書からの削除
- [x] 通常のトークン識別子がマクロであれば置換リストに展開する処理の実装
- [x] 展開の無限ループ防止と循環参照のハンドリング機構（hideset）

## 4. 条件付き取り込み・条件分岐 (`#if` 系)
- [x] 現在の条件分岐の有効/無効状態とネスト深さを管理するスタック（コンテキスト）の実装
- [x] `#if`, `#ifdef`, `#ifndef` のパースと条件評価
- [x] `#else`, `#elif` のパースと分岐切り替え
- [x] `#endif` のパースと状態のポップ
- [x] スキップ状態（無効ブロック）にあるトークン列を探査・除外する機能
- [x] `#if` 用の定数式評価パーサ（既存の `expr` ベース等）の実装

## 5. 関数マクロ (`#define MACRO(a, b)`)
- [x] カッコ付きパラメータのパースと保存
- [x] マクロ呼び出し時の実引数トークン列の収集機能
- [x] マクロの置換リスト内の仮引数を実引数へ置換（適用）するマクロ展開エンジンの拡張
- [x] E2Eテストへの複雑なマクロ関数・再帰的展開などのテストケース追加

## 6. その他ディレクティブ・細かな仕様準拠
